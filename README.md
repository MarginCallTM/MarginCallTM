### 🔄 [Push_swap](https://github.com/acombier/push_swap)
Efficient stack sorting algorithm with multiple optimization strategies
- **Simple algo:** O(n²) for small datasets (2-5 elements)
- **Chunk algo:** O(n√n) for medium datasets (20-100 elements)
- **Radix algo:** O(n log n) for large datasets (100-500 elements)
- Built-in benchmarking system for performance analysis
- Achieves <700 operations for 100 numbers, <5500 for 500 numbers

**Tech:** C, Algorithm Design, Performance Optimization
